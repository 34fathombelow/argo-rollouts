    Rollouts Create¶

    Create a Rollout, Experiment, AnalysisTemplate, or AnalysisRun resource

    Synopsis¶

    This command creates a new Rollout, Experiment, AnalysisTemplate, or AnalysisRun resource from a file.

    kubectl argo rollouts create [flags]

    Examples¶

    # Create an experiement and watch it
kubectl argo rollouts create -f my-experiment.yaml -w

    Options¶

      -f, --filename stringArray   Files to use to create the resource
  -h, --help                   help for create
      --no-color               Do not colorize output
  -w, --watch                  Watch live updates to the resource after creating
